# app-09

基于 .NET 7 + ASP.NET Core Web API + EF Core + MySQL + Redis 构建的知识库管理系统。

## 项目结构

- KnowledgeBase.API: ASP.NET Core Web API 入口
- KnowledgeBase.Application: 应用服务层
- KnowledgeBase.Domain: 领域实体与接口
- KnowledgeBase.Infrastructure: 基础设施层（EF Core、Redis、安全）

## 技术栈

- .NET 7
- ASP.NET Core Web API
- Entity Framework Core + Pomelo.MySql
- JWT 认证
- Redis 缓存
- Swagger/OpenAPI
